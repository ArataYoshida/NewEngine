# NewEngine

## 日本語

NewEngine は、公式バイナリとして配布される Windows 向けゲーム制作環境です。
このリポジトリは配布専用で、Editor や Engine のソースコードは含みません。

### ダウンロードとインストール

通常は NewLauncher から最新版を自動取得してインストールしてください。
手動で確認する場合は、このリポジトリの Releases にある最新リリースを参照してください。

リリースには次のファイルが含まれます。

- `engine-manifest.json`
- `NewGameEngine-<version>-win-x64.zip`

### ソースコードについて

このリポジトリには NewEngine、NewEditor、NewLauncher、NewScriptCore などのソースコードは公開しません。
公開されるのは、ユーザーが Editor を起動してプロジェクト制作に使うための公式バイナリのみです。

### ライセンス概要

未改造の公式バイナリは商用プロジェクトにも使用できます。
ただし、バイナリの改造、リバースエンジニアリング、再配布は禁止です。
詳細は `LICENSE` を確認してください。

## English

NewEngine is a Windows game creation environment distributed as official binary releases.
This repository is release-only and does not contain the source code for the Editor or Engine.

### Download and Installation

Use NewLauncher to automatically fetch and install the latest release.
If you need to inspect releases manually, open the latest item in this repository's Releases page.

Each release includes:

- `engine-manifest.json`
- `NewGameEngine-<version>-win-x64.zip`

### Source Code

This repository does not publish source code for NewEngine, NewEditor, NewLauncher, NewScriptCore, or related private components.
Only official binaries needed to run the Editor and build projects are distributed here.

Modification, reverse engineering, and redistribution of the binaries are not allowed.
